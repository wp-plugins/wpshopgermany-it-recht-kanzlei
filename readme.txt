== Description ==

Mit Hilfe dieses Plugins ist es möglich deutsche rechtssichere Texte  (für Shops) zu aktualisieren. Die Texte werden von der IT-Recht Kanzlei München bezogen (Abo vorausgesetzt).

Das Plugin wurde für das Wordpress Plugin wpShopGermany entwickelt. Es kann auch ohne das Shop Plugin genutzt werden. 

Um diesen Dienst nutzen zu können ist ein Account bei der IT-Recht Kanzlei München nötig.

== Installation ==

Zur Installation beachten Sie bitte folgende Schritte:

1. Die Plugindateien nach wp-content/plugins/wpshopgermany-itrecht kopieren, oder über das Wordpress Backend: Plugins > Installieren > Hochladen die ZIP Datei hochladen.
2. Das Plugin im Plugin Menü von Wordpress aktivieren.
3. Seiten für die Texte anlegen. (Im Wordpress Menü "Seiten")
4. Die Accountdaten von der IT-Recht Kanzlei München in die Einstellungen unter Einstellungen -> IT-Recht Kanzlei eintragen.
5. Die Seiten auswählen auf denen die Texte von der IT-Recht Kanzlei eingesetzt werden sollen, oder neue anlegen.
5. Button "Speichern" betätigen.

Die API-URL aus den Einstellungen und auch der API-Key müssen auf Seiten von der IT-Recht Kanzlei hinterlegt werden, damit der Abgleich erfolgen kann.
